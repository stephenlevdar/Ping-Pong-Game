# MY PONG GAME
This is two-player Ping Pong Game I built with Pygame library! Control paddles to bounce the ball and score points-first to 5 wins.
It has music and bounds sound tooooooo-_-
---
## Features                                                         
- Two paddles: Left uses W/S, right uses UP/DOWN arrow.
- Ball bounces off walls and paddles.
- Scoring: Miss the ball to give a point to the opponent
- Random ball direction after each point
- Background music that loops.
- Bounce sound effect when the ball hi8ts a paddle.
- win message when someone reaches 5 point.
